# full-amazon-clone
This project is a Full Amazon Clone developed using React.js and MongoDB, replicating essential features of an e-commerce platform. It supports user authentication, product browsing, cart management, and secure order handling. The application follows modern web development practices and is designed to be scalable and responsive.
